# Sequelize 查询 mysql 数据库
## 查全部

```
User.findAll();
// sql: SELECT * FROM `user`;
```

## 查询指定字段数据

```
User.findAll({
    'attributes': ['id', 'name']
});
// sql: SELECT `id`, `name` FROM `user`;
```

## 分页查询并且获取总数

```
User.findAndCountAll({
    'attributes': ['id', 'name'],
    limit: 20, // 每页多少条
    offset: 0 // 跳过多少条
})
// sql: 
// SElECT count(*) AS `count` FROM `user`;
// SELECT `id`, `name` FROM `user` LIMIT 20, 0;
```

## where 查询条件
#### IN 语句

```
User.findAll({
    where：{
        id: [2, 3, 4]
    }
});

User.findAll({
    where：{
        id: {
            '$in': [2, 3, 4]
        }
    }
});

// sql
SELECT * FROM `user` WHERE `user`.`id` IN (1, 2, 3);
```

#### AND 语句

```
User.findAll({
    where：{
        id: 1,
        name: "a"
    }
});

User.findAll({
    where：{
        id: {
            '$and': [
                {id: 1},
                {name: "a"}
            ]
        }
    }
});

// sql
SELECT * FROM `user` WHERE `user`.`id` = `1` AND `user`.`name` = `a`;
```

#### 其他运算符

```
User.findAll({
    where：{
        id: {
            '$eq' : 1, //id = 1
            '$ne' : 1, //id != 1
            '$gt' : 1, //id > 1
            '$gte' : 1, //id >= 1
            '$lt' : 10, //id < 10
            '$lte' : 10, //id <= 10
            '$between' : [1, 10], //id BETWEEN 1 and 10 （相当于 id >= 1 id <= 10）
            '$notBetween' : [1, 10], //id NOT BETWEEN 1 and 10
            '$notIn': [3, 4], //id NOT IN (3, 4)
            '$like': '%1%' // LIKE '%1%'（左右模糊查询）
            '$notLike': '%1%' // NOT LIKE '%1%'（左右模糊查询）
        }
    }
});
```
