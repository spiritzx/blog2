# 遮罩层镂空
## 需求
在遮罩层中间镂空一个空间

## 实现思路
遮罩层不在使用

`background：rgba(0,0,0,0.6);`

使用
`box-shadow:0,0,1500,rgba(0,0,0,0.6)`;

## 代码
```
html:
<div class="base">
    <div class="mask"></div>
</div>

css:
.base {
    height: 100%;
    width: 100%;
}
// 矩形镂空
.mask {
    width: 100%;
    height: 225px;
    box-shadow:0,0,1500,rgba(0,0,0,0.6);
}
// 圆形镂空
.mask {
    width: 300px;
    height: 300px;
    border: 50%;
    box-shadow:0,0,1500,rgba(0,0,0,0.6);
}
```
