
# useRef
> 通过 ref 获得组件和 DOM 元素的属性和方法

## 使用

```
function Parent() {
    let dom = useRef(null);
    let comp = useRef(null);
    return (
        // 绑定 dom 元素
        <button ref={dom} onClick={() => {setNameFn()}} id="Button">修改</button>
        // 绑定组件
         <ChilderThree ref={comp} name = {name} setNameFn={setNameFn} />
    )
}
```
