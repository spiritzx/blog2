
# 浅谈函数防抖与节流 
## 函数防抖
> 在一个连续的事件触发或动作中，只生效一个，其他屏蔽

### 后置防抖
```
/*
    * 后置防抖，一段时间内连续触发事件，最后触发的事件才生效，如页面大小的拖拽，页面滚动等
    * @params {fn}: 事件触发的函数
    * @params {wait}: 延迟时间，屏蔽掉该延迟时间的函数
    * @return {function}: 延迟时间，屏蔽掉该延迟时间内的事件
*/
function afterDebounce(fn, wait) {
    let timeout
    if (typeof fn != 'function') {
        throw new TypeError('Expected a function')
    }
    return function () {
        let context = this; //传给目标函数
        clearTimeout(timeout);
        timeout = setTimeout(
            () => { fn.apply(context, arguments) } //修复
            , wait)
        }
}
```

### 前置防抖

```
/*
    * 前置防抖，一段时间内连续触发事件，第一次触发的事件才生效，其他无效，应用于按钮请求后台，防止短时间多次请求
    * @params {fn}: 事件触发的函数
    * @params {wait}: 延迟时间，屏蔽掉该延迟时间的函数
    * @return {function}: 延迟时间，屏蔽掉该延迟时间内的事件
*/
function beforeDebounce(fn, wait) {
    let timeout
    if (typeof fn != 'function') {
        throw new TypeError('Expected a function')
    }
    return function () {
        let context = this; //传给目标函数
        // 将 timeout 急定时器 id 作为判断值
        if (!timeout) {
            fn.apply(context, arguments)
        }
        clearTimeout(timeout);
        timeout = setTimeout(
            () => { timeout = null} //修复
            , wait)
        }
}

```


## 函数节流
> 在一个连续的事件触发或动作中，按照延迟时间规律触发，其他屏蔽

```
export function throttle(fn: Function, delay = 500): Function{
    let timeId: NodeJS.Timeout|null = null;
    return function (...arg) {
        if (timeId === null) {
            timeId = setTimeout(() => {
                fn.call(this, ...arg);
                clearTimeout(timeId as NodeJS.Timeout);
                timeId = null;
            }, delay)
        } 
    }
}
```
