
# js基础: 变量回收机制
> 从一个问题来了解 js 变量回收机制

## 问题
以下代码,哪些变量没被回收

```
var a = 1;
var a = 2;
var add = (function() {
  var c = 1;
  return function () {
    return a++;
  }
})()
add()
```

## 解析
变量回收的几种情况:
1. 全局变量不会被回收
2. 函数运行完,函数内部局部变量会随着函数销毁而被回收
3. 被一个其他作用域引用着的变量不会被回收

根据以上情况, 变量 a 和 add 不会被回收,而局部变量 c 被 函数 add 引用,所以也不会被回收,从而结果为变量 a add c 不会被回收
