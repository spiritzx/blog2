
# 常量与指针
> 常量与指针的三种情况

## 常量指针
> 锁死内存空间及锁死值----可以改变指针的地址，不能改变值

```
int a = 10;
int b = 10;
const int * p;
p = &a;

*p = 20; // 错误，不能改变值
*P = &b; // 正确
```

## 指针常量
> 锁死内存地址----可以改变值，不能改变指针的地址

```
int a = 10;
int b = 10;
const int * p;
p = &a;

*p = 20; // 正确，可以改变值
*P = &b; // 错误，不能改变地址
```

## 常量指针与指针常量
> 锁死内存地址和指针的地址 ---- 值和地址都不能改变

```
int a = 10;
int b = 10;
const int * const p;
p = &a;

*p = 20; // 错误，不可以改变值
*P = &b; // 错误，不能改变地址
```
