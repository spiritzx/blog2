
# dart 常用数据类型转换

## 转化布尔值

```
Bool flag = 0.1 == 0.2; // false
```

## 字符串转数值

```
String str = "1";
double num0 = double.parse(str); // 1.0
int num1 = int.parse(str); // 1
```

## 数值转字符串

```
String str = 0.toString(); // str
```

## 数组转字符串

```
Sting str = [1, 2, 2].join("");
```

## List 转 Set

```
List<int> arr1 = [1, 2, 3, 1];
Set<int> set = arr1.toSet();
```

## Set 转 List

```
List<int> arr1 = [1, 2, 3, 1];
Set<int> set = arr1.toSet();
List<int> arr2 = set.toList()
```
