# vscode 版本1.44 说明文件翻译
> March 2020 (version 1.44)

## 更新的要点
1. 可访问性的改进：轻松分别视图导航，设计 UI 简洁权限设置
2. 时间线展示：可以在 vscode 查看像Git提交历史记录这样的时间序列事件
3. 更快的打开文件：通过文件查找（Ctrl+P）打开文件更加顺滑，通过文件标题筛选文件
4. 扩展包展示：快速查看扩展包的捆绑了什么
5. 保存 撤销/重做 堆栈功能：为了重新打开文件保留 撤销/重做 堆栈功能
6. 远程开发：直接检查拉入容器的请求，为 Kubernetes 提供支持
7. 设置同步的预览：设置同步现在允许你在机器之间共享代码片段和用户界面状态
8. 新的 python 教程：为创建 python 容器和构建科学数据模型新增了教程
