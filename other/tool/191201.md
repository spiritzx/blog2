## one: 表格语法

```
aa|bb|cc
:-|:-|:-
11|22|33
```

1. 左对齐: “ :- ”
2. 居中：“ -:- ”
3. 右对齐：“ -: ”
4. 第二行的不能省
```
:-|:-|:-
```

## two: 通过瞄点生成目录
### 生成锚点
1. 第一种|

```
## one
```

会翻译成

```
<h2 id="one">one<h2>
```

2. 第二中直接写出

```
<h2 id="one">one<h2>
```

### 访问锚点

```
[one](#one)
```
